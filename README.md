# Azhar General Score Converter

A simple Python program that converts scores between Al-Azhar and General Education.

## Features

- Convert Azhar score to General score.
- Convert General score to Azhar score.
- Simple command-line interface.
- Beginner-friendly Python code.

## How to Run

```bash
python azhar_general_score_converter.py
```

## Example

```
1 - Azhar to General
2 - General to Azhar

Choose: 1
Enter Azhar Score: 450

Equivalent General Score = 247.06
```
